# Shopping Merchant 购物商家选择助手

帮你判断在淘宝、京东、天猫、拼多多、抖音等平台该相信哪种店铺——旗舰店、自营、授权店还是第三方。

## What it does
- 按商品品类评估风险等级（低/中/高）
- 对比不同卖家类型的正品保障、售后和退换政策
- 在一分钱一分货的权衡中给出信任度调整后的推荐
- 解释"便宜"背后的风险成本

## Example scenarios

**1. 电子产品：京东自营 vs 第三方**
> 👤 同一款耳机，京东自营贵 80，第三方便宜，怎么选？
> 🤖 先说结论：电子产品属高风险品类，京东自营的保修和退换货兜底值这 80 块。第三方低价背后可能是水货、翻新或售后推诿的风险。除非那家是品牌认证旗舰店且评价稳定，否则多花 80 买省心。

**2. 护肤品选店类型**
> 👤 护肤品买旗舰店、授权店还是普通店？
> 🤖 化妆品是最怕假货的品类，直接选官方旗舰店最安全。授权店如果平台有认证标识也可以接受；普通第三方店铺在这个品类风险太高，不建议省那几十块。正品和脸，你只能冒一个险。

**3. 工厂店 vs 旗舰店的价差**
> 👤 拼多多工厂店比天猫旗舰店便宜很多，风险在哪里？
> 🤖 风险主要在三个地方：正品确定性（工厂店可能有批次差异）、售后响应速度、退换货便利性。低价日用品（收纳盒、手机壳）可以走拼多多工厂店；高价值或对品质敏感的商品，那多出来的钱买的是确定性。
